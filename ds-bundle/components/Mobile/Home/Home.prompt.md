# Home — الرئيسية

A calm, scholarly landing screen for the student-facing mobile app. Single vertical scroll on warm sand (`#f3ecdd`), 22px side padding, 118px bottom padding to clear the fixed mini-player.

## Sections, top → bottom
1. **Header** — 40px logo circle (brass ring + teal rhombus), title "رِواق العِلم" (Amiri 22/700, teal) + subtitle "مجالس الدروس الشرعية" (11px ghost), search icon-button on the left (`#e9e0cd`, radius 13).
2. **"أكمِل الاستماع" feature card** — teal `#1f4a42`, radius 22, faint concentric-circle motif. 62px artwork tile, brass eyebrow, Amiri 19/700 title in `#f6f0e2`, sheikh line. Progress row: `١٨:٤٢` / 62% brass track / `٣٠:١٥` + 42px brass play/pause button. Tapping toggles the shared `playing` state.
3. **"أُضيف حديثاً"** — section header + "عرض الكل", horizontal snap scroller of 158px cards (tinted gradient cover + play glyph + duration chip, Amiri 15/700 title, ghost sheikh).
4. **"الأقسام العِلمية"** — 2-column grid, 12px gap. Each card: 42px letter tile (Amiri teal letter) + name + lesson count.
5. **"لا تنسَ الدعاء"** — quiet dashed-brass card inviting du'a for the platform's scholars.
6. **Mini player (fixed)** — pinned 14px from bottom, teal `#16352f`, radius 20, 40px artwork + title + 62% progress + 40px brass play/pause. Shares `playing` with the feature card.

## Notes
- All numerals Arabic-Indic; times use tabular figures.
- Two background colors only: sand for the page, teal for feature/player surfaces. No gamification, no bright colors.
- This card shows the **playing** state (pause bars in both play buttons).
